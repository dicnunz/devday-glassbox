Glassbox: The One-Prompt Escape

Room 1/5: Verify

Collect source seals. Avoid unverified claims.

Receipt: $0 spend, static assets only, no tracking.

How Built

Glassbox is a static HTML/CSS/JS canvas game built end-to-end in Codex with GPT-5.5. Original visual assets were generated during this task with Image Gen. It uses no backend, no runtime API calls, no accounts, no tracking, and no paid services.

Goal: move the cursor-agent through Verify, Generate, Build, Deploy, and Submit while avoiding claims, paid APIs, card entry, and browser-window hazards.
