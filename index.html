How Built

Glassbox is a static canvas puzzle built in Codex with GPT-5.5. The art set was generated as separate Image Gen assets: key art, board texture, player object, core, prism, hazard, gate, room card, and victory preview.

The generated images are atmosphere and object references. The seven handcrafted levels, collision, beam branching, mirror/prism behavior, shutters, lenses, gate sensors, UI, and win logic are deterministic JavaScript.

No backend, no runtime API calls, no accounts, no tracking, no paid assets, and no card entry. Cost: $0.00.
